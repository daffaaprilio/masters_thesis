\section{dPCD Gene Module (Literature Curation)}
\hspace{1em}The dPCD (developmentally programmed cell death) module used as the query set for the D-gene positive control (Section~3.2) was assembled in two tiers. First, a small set of conserved dPCD indicator genes was taken directly from \citeauthor{olvera-carrillo_conserved_2015}'s literature review of PCD marker genes conserved across plant developmental and environmental cell-death programmes \cite{olvera-carrillo_conserved_2015}, with sorghum orthologs identified manually (Table~\ref{tab:dpcd-conserved}). Second, this core set was expanded to a broader 66-gene module (Table~\ref{tab:dpcd-broad}) by mapping every \textit{Arabidopsis thaliana} ortholog family member to \textit{Sorghum bicolor} via Ensembl Plants BioMart, then resolving each SORBI stable ID to its EGI (\texttt{LOC}-prefixed) identifier using the NCBI \texttt{gene\_info} table for \textit{Sorghum bicolor} (taxonomy ID 4558); 113 of 125 SORBI IDs mapped to an EGI, yielding 66 unique sorghum genes across 33 distinct AGI ortholog groups. This 66-gene module is the one used throughout the enrichment tests and co-expression analysis in Section~3.2.
